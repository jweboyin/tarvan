# Popup 弹出层

### 介绍

弹出层容器，用于展示弹窗、信息提示等内容，支持多个弹出层叠加展示。

## 代码演示

### 引入

```jsx
import { Popup } from 'czh-weapp'
```

### 基础用法

通过show属性控制弹出层是否展示。

<Image src="/imgs/popup/show.jpg" />

```jsx
<Popup 
  transition="scale"
  show
>
  <view>内容</view>
</Popup>
```

### 弹出位置

通过position属性设置弹出位置，默认居中弹出，可以设置为top、bottom、left、right。

<Image src="/imgs/popup/position.jpg" />

```jsx
<Popup 
  position="top"
  show
/>
```

### 关闭图标

设置closeable属性后，会在弹出层的右上角显示关闭图标，并且可以通过closeIcon属性自定义图标，使用closeIconPosition属性可以自定义图标位置。

<Image src="/imgs/popup/icon.jpg" />

```jsx
<Popup 
  position="bottom"
  closeIconPosition="top-left"	
  show
  closeable
/>	
```

### 圆角弹窗

设置round属性后，弹窗会根据弹出位置添加不同的圆角样式。

<Image src="/imgs/popup/round.jpg" />

```jsx
<Popup 
  position="bottom"
  show
  round
/>		
```

## API

### Props

<div class="table-container">
| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| show | 是否显示弹出层	 | _boolean_ | `false` |
| zIndex | z-index 层级	 | _number_ | `100` |
| duration | 动画时长，单位为毫秒 | _number_ | `300` |
| lockScroll | 是否锁定背景滚动 | _boolean_ | `true` |
| closeable | 是否显示关闭图标	 | _boolean_ | `false` |
| closeIcon | 关闭图标名称 ｜ 图片文件名 ｜ 图片链接	| _string_ | `cross` |
| closeIconPosition | 关闭图标位置，可选值为 top-left ｜ bottom-left ｜ bottom-right	| _string_ | `top-right` |
| position | 弹出位置，可选值为 top ｜ bottom ｜ right ｜ left| _string_ | `center` |
| round | 是否显示圆角	 | _boolean_ | `false` |
| overlay | 是否显示遮罩层	| _boolean_ | `true` |
| closeOnClickOverlay | 是否在点击遮罩层后关闭	| _boolean_ | `true` |
| safeAreaInsetBottom | 是否为iPhone留出底部安全距离	| _boolean_ | `true` |
| safeAreaInsetTop | 是否留出顶部安全距离（状态栏高度）	| _boolean_ | `false` |
</div>

| onClose | 关闭弹出层时触发	| - |

| onEnter | 进入中触发	 | - |
| onAfterEnter | 进入后触发		 | - |
| onBeforeLeave | 离开前触发			 | - |
| onLeave | 离开中触发					 | - |
| onAfterLeave | 离开后触发				 | - |
