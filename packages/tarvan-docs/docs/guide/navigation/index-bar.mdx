# IndexBar 索引栏 

### 介绍

用于列表的索引分类显示和快速定位。

## 代码演示

### 引入

```jsx
import { IndexBar } from 'czh-weapp'
```

### 基础用法

点击索引栏时，会自动跳转到对应的锚点位置。

<Image src="/imgs/index-bar/basic.jpg" />

```jsx
<IndexBar>
	<IndexBar.Group index="A">
		<Cell title="文本" />
		<Cell title="文本" />
		<Cell title="文本" />
	</IndexBar.Group>

	<IndexBar.Group index="B">
		<Cell title="文本" />
		<Cell title="文本" />
		<Cell title="文本" />
	</IndexBar.Group>

	<IndexBar.Group index="C">
		<Cell title="文本" />
		<Cell title="文本" />
		<Cell title="文本" />
	</IndexBar.Group>

	...
</IndexBar>
```

### 开启延迟渲染

通过lazyRender属性开始延迟渲染，优化长列表性能

```jsx
<IndexBar lazyRender>
	....
</IndexBar>
```

### 自定义侧边栏索引项样式

通过className或style属性设置css变量来控制索引项的样式。具体支持的样式变量请看下列对应的文档。

<Image src="/imgs/index-bar/custom.jpg" />

```jsx
<IndexBar 
	style={{
		'--czh-index-bar-index-highlight-color': '#FFFFFF',
		'--czh-index-bar-index-highlight-background-color': '#07c160'
	}}
>
	....
</IndexBar>
```

## API

### Props

<div class="table-container">
| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| sticky | 是否开启锚点自动吸顶	 | _boolean_ | `true` |
| stickyOffsetTop | 锚点自动吸顶时与顶部的距离	 | _number_ | `0` |
| safeAreaBottom | 是否开启底部安全区域 | _boolean_ | `true` |
| showBackOffsetTop | 是否显示返回顶部按钮 | _boolean_ | `true` |
| disableSidebar | 是否禁用侧边栏 | _boolean_ | `false` |
| lazyRender | 是否开启延迟渲染, 优化长列表渲染性能 | _boolean_ | `false` |
| scrollDuration | 滚动动画时长, 毫秒 | _number_ | `100` |
</div>

### Events

| 事件名 | 说明 | 参数  |
| --- | --- | --- |
| onSelect | 选中字符时触发		 | index: 索引字符 |
