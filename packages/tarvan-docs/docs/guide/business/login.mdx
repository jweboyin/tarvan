# Login 登录

### 介绍

小程序应用默认的用户登录界面，应用如无特殊设计要求，使用该组件即可。

## 代码演示

### 引入

```jsx
import { Login } from 'czh-weapp'
```

### 基础用法

默认调用的是账号中心服务的登录接口，如应用有特殊登录逻辑，可通过onLogin以及onLoginByWxMobile属性传入处理函数。

<Image src="/imgs/login/basic.jpg" />


```jsx
import { useMemoizedFn } from 'ahooks'

const onLogin = useMemoizedFn(async (username, password) => {
  // 处理账号密码登录
})

const onLoginSuccess = useMemoizedFn(async () => {
  // 处理登录成功以后
})

const onLoginByWxMobile = useMemoizedFn(async (wxMobileCode) => {
  // 处理微信手机号快捷登录
})

const onTapProtocol = useMemoizedFn(() => {
  // 处理点击用户协议
})

<Login
  agreementTitle="吾行养车用户协议及隐私政策" 
  onLogin={onLogin}
  onLoginByWxMobile={onLoginByWxMobile}
  onTapProtocol={onTapProtocol}
  onLoginSuccess={onLoginSuccess}
/>
```
