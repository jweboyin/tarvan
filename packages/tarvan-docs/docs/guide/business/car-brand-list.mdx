# CarBrandList 车辆品牌列表

### 介绍

用于显示一个车辆品牌列表。 

## 代码演示

### 引入

```jsx
import { CarBrandList } from 'czh-weapp'
```

### 基础用法

<Image src="/imgs/car-brand-list/basic.jpg" />

```jsx
<CarBrandList />
```

### 自定义品牌图标大小

通过className或者style属性改变css变量来自定义品牌图标大小。

<Image src="/imgs/car-brand-list/custom-icon.jpg" />

```jsx
<CarBrandList 
  style={{
    '--czh-car-brand-index-bar-item-icon-width': '30px',
    '--czh-car-brand-index-bar-item-icon-height': '30px'
  }}
/>
```

### 自定义品牌字体颜色

通过className或者style属性改变css变量来自定义品牌字体颜色。

<Image src="/imgs/car-brand-list/color.jpg" />

```jsx
<CarBrandList 
  style={{
   '--czh-car-brand-index-bar-item-color': '#00b578',
  }}
/>
```

### 点击品牌

通过onSelected属性来处理点击品牌事件，参数为车辆品牌ID。

<Image src="/imgs/car-brand-list/selected.jpg" />

```jsx
import Taro from '@tarojs/taro'
import { useMemoizedFn } from 'ahooks'

const onSelected = useMemoizedFn((brandId) => {
  Taro.showToast({
    title: `选中的车辆品牌ID: ${brandId}`,
    icon: 'none',
    duration: 3000
  })
})

 <CarBrandList onSelected={onSelected} />
```
